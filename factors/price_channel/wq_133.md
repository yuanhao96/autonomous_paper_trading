# WQ-133: Range rank minus return rank over a 20-day window

## Formula
ts_rank(high - low, 20) - ts_rank(delta(close, 1), 20)

## Interpretation
Range rank minus return rank over a 20-day window. High when today's range is historically wide but the return is historically modest -- signals potential reversal from range expansion without directional follow-through.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| window | 20 | [10, 40] |

## Category
price_channel
