# WQ-009: If all recent daily changes are positive (uptrend), follo...

## Formula
where(ts_min(delta(close, 1), 5) > 0, delta(close, 1), where(ts_max(delta(close, 1), 5) < 0, delta(close, 1), -1 * delta(close, 1)))

## Interpretation
If all recent daily changes are positive (uptrend), follow momentum. If all negative (downtrend), follow. Otherwise, reverse. Trend vs reversal conditional.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| window | 5 | [2, 10] |

## Category
volume_price
