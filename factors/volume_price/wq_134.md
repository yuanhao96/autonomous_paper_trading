# WQ-134: 12-day return scaled by relative volume

## Formula
delta(close, 12) / delay(close, 12) * volume / sma(volume, 20)

## Interpretation
12-day return scaled by relative volume. Medium-term momentum confirmed by above-average volume.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| return_window | 12 | [6, 24] |
| vol_sma | 20 | [10, 40] |

## Category
volume_price
