# WQ-015: Volume-correlation-weighted average of daily returns over...

## Formula
-1 * sum(delta(close, 1) * correlation(close, volume, 10), 3) / sum(correlation(close, volume, 10), 3)

## Interpretation
Volume-correlation-weighted average of daily returns over 3 days.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| return_window | 3 | [1, 6] |
| corr_window | 10 | [5, 20] |

## Category
momentum
