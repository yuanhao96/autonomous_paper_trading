# WQ-135: 5-day return minus 20-day return

## Formula
delta(close, 5) / delay(close, 5) - delta(close, 20) / delay(close, 20)

## Interpretation
5-day return minus 20-day return. Short-term momentum relative to longer-term -- momentum acceleration.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| short_window | 5 | [2, 10] |
| long_window | 20 | [10, 40] |

## Category
momentum
