# WQ-076: Volatility (stddev of returns) minus average absolute return

## Formula
stddev(delta(close, 1) / delay(close, 1), 20) - sma(abs(delta(close, 1) / delay(close, 1)), 20)

## Interpretation
Volatility (stddev of returns) minus average absolute return. Positive when return distribution is fat-tailed relative to mean movement.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| window | 20 | [10, 40] |

## Category
volatility
