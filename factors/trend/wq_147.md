# WQ-147: Information ratio / Sharpe-like measure over 20 days

## Formula
sma(delta(close, 1) / delay(close, 1), 20) / stddev(delta(close, 1) / delay(close, 1), 20)

## Interpretation
Information ratio / Sharpe-like measure over 20 days. Mean return divided by return volatility -- reward-to-risk trending signal.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| window | 20 | [10, 40] |

## Category
trend
