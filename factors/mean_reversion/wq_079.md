# WQ-079: 3-day return adjusted by volatility

## Formula
delta(close, 3) / delay(close, 3) + sign(delta(close, 3)) * stddev(close, 20) / close

## Interpretation
3-day return adjusted by volatility. Penalizes large moves -- mean reversion in volatile conditions.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| return_window | 3 | [1, 6] |
| vol_window | 20 | [10, 40] |

## Category
mean_reversion
