# WQ-010: Ranked conditional momentum/reversal

## Formula
rank(where(ts_min(delta(close, 1), 4) > 0, delta(close, 1), where(ts_max(delta(close, 1), 4) < 0, delta(close, 1), -1 * delta(close, 1))))

## Interpretation
Conditional signal: if all recent daily changes were positive (momentum up), use today's change; if all negative (momentum down), use today's change; otherwise reverse today's change (contrarian). Cross-sectional ranking compares this signal across tickers.

|-------|---------|-------|
| lookback | 4 | [2, 8] |
