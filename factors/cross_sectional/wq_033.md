# WQ-033: Ranked intraday return

## Formula
rank(-1 * (1 - open / close))

## Interpretation
Ranks the open-to-close return across tickers. The negative sign means stocks that fell from open to close rank higher. A simple cross-sectional contrarian intraday signal.

## Parameters
None

## Type
cross_sectional

## Category
cross_sectional
