# WQ-001: Ranked conditional volatility argmax

## Formula
rank(ts_argmax(SignedPower(where(returns < 0, stddev(returns, 20), close), 2), 5)) - 0.5

## Interpretation
Ranks the day-of-max of a conditional measure: uses rolling volatility when returns are negative, raw close otherwise. SignedPower squares the magnitude while preserving sign. Cross-sectional ranking identifies relative positioning.

## Parameters
| Param | Default | Range |
|-------|---------|-------|
| vol_lookback | 20 | [10, 30] |
| argmax_lookback | 5 | [3, 10] |

## Type
cross_sectional
